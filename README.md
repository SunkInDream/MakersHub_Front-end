# MakersHub_Front-end

#### 版本命名规则及近期开发安排
v0.1 实现基本框架、宣传信息可见，实现宣传部工作台。
v0.2 实现借物等基管功能可用（以及开门这个重要功能），实现基管工作台。
v0.3 实现项目立项等项目相关功能、积分兑换等可用，实现项目工作台和运维工作台可用。
v1.0 实现除社区外所有功能可用。
